# 📝 Weather App

A simple and interactive **Weather Forecast web application** built using **JavaScript, API integration, DOM manipulation, and Bootstrap**.
This project helps users check real-time weather conditions quickly with a clean, responsive, and modern UI.

---

## 🚀 Live Demo 🔗 [Go_WeatherApp](https://weatherapp11224.netlify.app/)

## 📌 Features

* 🌍 Search weather by city name
* 🌡️ Display real-time temperature and weather conditions
* 💨 Show humidity, wind speed, and additional weather details
* 📱 Fully responsive UI using Bootstrap
* ⚡ Fast performance with pure JavaScript and API integration

---

## 🛠️ Technologies Used

* **HTML5** – Structure of the application
* **CSS3** – Custom styling
* **Bootstrap** – Responsive design and UI components
* **JavaScript (ES6)** – Application logic
* **Weather API** – Fetching real-time weather data
* **DOM Manipulation** – Dynamic rendering of weather information
* **Fetch API** – Handling API requests efficiently
